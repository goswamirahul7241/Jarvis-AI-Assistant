# Jarvis AI Assistant

## Project Overview
Jarvis AI Assistant is a sophisticated voice assistant designed to help users perform tasks efficiently and effectively. The application utilizes modern machine learning algorithms and natural language processing techniques to provide accurate responses and assistance.

## Features
- **Voice Recognition**: Understands and processes user queries through natural language.
- **Task Management**: Assists users in managing their daily tasks and reminders.
- **Information Retrieval**: Quickly provides information from various sources, including the web.
- **Integration**: Connects with third-party applications and services to enhance functionality.

## Setup Instructions
1. Clone the repository: `git clone https://github.com/goswamirahul7241/Jarvis-AI-Assistant`
2. Kilo CLI: `install kilo cli on your device window/mac`
3. Pocket-TTS server: install pocker-tts using commond `uvx pocket-tts serve`
4. Navigate to the project directory: `cd Jarvis-AI-Assistant`
5. Install required dependencies: `pip install -r requirements.txt`
6. Run the application: `python jarvis.py`
7 for run voice make sure your pocket-tts server is runnuing.

## Usage Guidelines
- Open command prompt and direct your jarvis file where you save then run jarvis.py file
- To start the assistant, simply execute the application and say "Jarvis, [your command]".
- For help, you can ask, "Jarvis, help with commands."  

Explore the capabilities of Jarvis and streamline your daily tasks with the advanced features of this AI assistant!
